Frequently Asked Questions

How do I start and stop the application?

To start the application: Double-click on the collect icon on your desktop. This will start the web server and open your browser.

To stop the application: Press CTRL+C in the Windows terminal window where the web server is running. This controlled shutdown prevents database corruption.

How do I update the application?

To get the newest version of the application, double-click on the sync collection icon on your desktop. This will pull the latest source code from GitHub.

Generally, Paul will contact you to tell you when new bug fixes or features are available.

Most code updates will be compatible with your existing database. Sometimes certain types of updates will require coordinated changes between the application and the underlying database. In these cases, Paul will schedule a maintenance time to make the coordinated changes.

How do I search for specific items?

There are several ways to find items in the collection:

  • Drill Down: Navigate from Countries down through the hierarchy
  • Direct ID Search: Use the green "Go" button in the upper right to directly access Load, Date, Variation, or Box records by ID (e.g., B231 for Box 231)
  • Headstamp Search: Use the blue "Search" button to find headstamps matching your query (either by headstamp code or name)
  • Filtered Lists: Use filters on list pages to narrow down results

How do I navigate between different calibers?

The application now supports multiple calibers (cartridge types). To switch between them:

  1. Click on the caliber dropdown in the top navigation bar
  2. Select the caliber you want to view
  3. You will be taken to the dashboard for that caliber

All navigation and searches will apply only to the currently selected caliber.

What if I need to move a record to a different parent?

If you need to correct a mistake or reorganize your collection:

  1. Go to the detail page for the record you want to move
  2. Click the "Move" button
  3. Select the new parent record from the options
  4. Confirm the move

The record and all its children will be moved to the new location in the hierarchy.
